## 어려웠던 부분

- gsap를 리액트에 적용하는 부분이 어려웠다.
  리액트에서 직접 DOM을 조작하기 위해서는
  useRef를 사용해야 한다는 것을 지나가듯이 배웠던 것 같은데,
  이번 기회에 조금 더 공부할 수 있었다.

```jsx
import { (기타 리액트 훅들), useRef } from "react"
...
const imgList = useRef();
...
<img ref={imgList}></img>
```

useRef를 변수에 할당해 주고,

가져오고자 하는 DOM 요소에 ref 속성으로 전달해 주면

해당 DOM을 가져올 수 있다.
