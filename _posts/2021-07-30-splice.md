<h3>splice method는 어떻게 사용하는 것일까?</h3>

```javascript
let array = ['A', 'C', 'D', 'E']
  array.splice(1, 0, B);

  // array = ['A', 'B', 'C', 'D', 'E']
```

원하는 index 자리(1)에, 원하는 자리에서부터 (0)개의 index를 삭제하고, 원하는 것 ('B')를 넣는다.
아래 그림을 보면 쉽게 이해할 수 있다.

다른 경우를 살펴보자.

```javascript
let array = ['A', 'B'. 'C', 'D', 'E']
  array splice(1, 1)
```

원하는 index와(1), 자신을 포함한 (1)개의 index를 삭제한다.
